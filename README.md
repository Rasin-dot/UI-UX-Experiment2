# UI-UX-Experiment2

## Aim:
To design a basic mobile and web layout for an existing case study using Figma by applying UI/UX design principles.

## Algorithm:
1. Open Figma and create a new design file.

2. Select the Android Small frame size for the mobile layout.

3. Design the Welcome Screen with college logo and Register/Login buttons.

4. Create the Sports Event Selection Screen displaying available events.

5. Design the Event Registration Form with input fields such as Full Name, Age, Gender, and Department.

6. Apply consistent colors, fonts, and button styles across all screens.

7. Use Gestalt principles like proximity and similarity to group related elements.

8. Add navigation flow between screens using prototype connections.

9. Design a Thank You screen with contact information.

10. Save and share the Figma design for submission.
